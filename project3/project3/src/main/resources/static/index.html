This API allows for the following Actions:

1. Adding / Reading / Updating Words In Translator Dictionary
2. Adding / Reading / Updating Languages In Translator Dictionary
3. Adding / Reading / Updating Targets for translation of words in Dictionary

Word Editing / Access

1. To Access list of all words, use the following url: http://csci3033.cs.mtsu.edu:8523/words


2. To access a word by the Id only, use the following url: http://csci3033.cs.mtsu.edu:8523/words/getById?id=$$
**enter requested word Id in place of $$**

Language Editing / Access

1. To Access list of all Languages, use the following url: http://csci3033.cs.mtsu.edu:8523/languages


2. To access a language by the Id only, use the following url: http://csci3033.cs.mtsu.edu:8523/languages/getById?id=$$
**enter requested Language Id in place of $$**

Target Editing / Access

1. To Access list of all Targets, use the following url: http://csci3033.cs.mtsu.edu:8523/Targets


2. To access a target by the Id only, use the following url: http://csci3033.cs.mtsu.edu:8523/targets/getById?id=$$
**enter requested Target Id in place of $$**
